# C6 HuTao Bot

C6 Hutao bot is a general use Discord bot developed by Matthew Kim

## Installation

<ins>will be underlined</ins>

```bash
pip install foobar
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
